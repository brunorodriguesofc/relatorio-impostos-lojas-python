📊 Relatório de Impostos por Loja em Python

Este projeto simula um sistema simples de análise de impostos de duas lojas, aplicando regras de negócio e organização de código com funções.

⸻

🚀 Funcionalidades
	•	Cálculo de imposto por loja
	•	Cálculo de imposto total consolidado
	•	Classificação de carga tributária
	•	Uso de funções para modularização
	•	Reutilização de lógica

⸻

📌 Regras de imposto
	•	Até 1000 → 10%
	•	Acima de 1000 → 15%
	•	Acima de 1500 → 20%

⸻

🧠 Conceitos aplicados
	•	Funções (def)
	•	Estruturas condicionais (if / elif / else)
	•	Listas
	•	Loop (for)
	•	Reutilização de código
	•	Separação de responsabilidades
